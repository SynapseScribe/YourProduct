## ✨ Features

- 🎨 **Modern Material You design** – dynamic colors, auto dark/light mode
- 🔄 **Fully customizable** – reorder, hide, or disable any AI assistant
- 📱 **Smooth tabbed browsing** – persistent sessions with background support
- 🔒 **Privacy-first** – blocks trackers & unnecessary connections by default
- 📏 **Adjustable font size** – read comfortably on any screen
- ⚡ **Lightweight & optimized** – fast, fluid performance
- 💾 **Persistent conversations** – survive app restarts
- 🎯 **Clean, intuitive interface** – zero distractions


## 🔒 Privacy & Data Collection

**AI Hub is fully open-source and collects absolutely nothing.**

- ✅ **Zero data collection** – no analytics, telemetry, crash reporters, or tracking
- ✅ **No personal information** – never asks for or stores name, email, or any identifying data
- ✅ **No user data** – never stores or sends any user data
- ✅ **Fully verifiable** – entire source code is public; audit or build it yourself


## 🛡️ Tracker & Ad Blocking

By default, AI Hub blocks **unnecessary connections** commonly used for tracking, analytics, and advertising. This significantly reduces what AI service websites can collect.

> ⚠️ **Important:** This is a best-effort measure, not a complete solution. Some tracking methods may still bypass these blocks. For maximum privacy, always review the privacy policies of the AI services you use.
